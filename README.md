# Ответ на пункт 2 численного метода

Сумма ряда Фурье связана с исходной функцией так

1. `S(x) = f(x)` во всех точках , кроме точек разрыва
2. В точках разрыва используем теорему Дирихле `S(x) = 1/2 * (f(x_0 + 0) + f(x_0 - 0))`

У меня в графиках это выражается в ступеньках и промежуточных значениях в точках разрыва
